API reference for Angular Material portal

import {PortalModule} from '@angular/material';

Directives

TemplatePortalDirective

Directive version of a TemplatePortal. Because the directive is a TemplatePortal, the directive instance itself can be attached to a host, enabling declarative use of portals.

Usage:

Hello {{name}}

Selector: [cdk-portal] [cdkPortal]
Exported as: cdkPortal

PortalHostDirective

Directive version of a PortalHost. Because the directive is a PortalHost, portals can be directly attached to it, enabling declarative use.

Usage:

Selector: [cdkPortalHost]

ComponentPortal

A ComponentPortal is a portal that instantiates some Component upon attachment.

Properties
NameDescription

component

The type of the component that will be instantiated for attachment.

injector

[Optional] Injector used for the instantiation of the component.

viewContainerRef

[Optional] Where the attached component should live in Angular's logical component tree. This is different from where the component renders, which is determined by the PortalHost. The origin is necessary when the host is outside of the Angular application context.

TemplatePortal

A TemplatePortal is a portal that represents some embedded template (TemplateRef).

DomPortalHost

A PortalHost for attaching portals to an arbitrary DOM element outside of the Angular application context.

This is the only part of the portal core that directly touches the DOM.
