API reference for Angular Material input

import {MatInputModule} from '@angular/material';

Directives

MdTextareaAutosize

Directive to automatically resize a textarea to fit its content.

Selector: textarea[mdTextareaAutosize]
Exported as: mdTextareaAutosize, matTextareaAutosize
Properties
NameDescription
@Input('mdAutosizeMaxRows')

maxRows

@Input('mdAutosizeMinRows')

minRows

Methods
resizeToFitContent

Resize the textarea to fit its content.

MdInput

Directive that allows a native input to work inside a MdFormField.

Selector: input[mdInput] textarea[mdInput]
Properties
NameDescription
@Input()

disabled

Whether the element is disabled.

@Input()

errorStateMatcher

A function used to control when error messages are shown.

@Input()

id

Unique id of the element.

@Input()

placeholder

Placeholder attribute of the element.

@Input()

required

Whether the element is required.

@Input()

type

Input type of the element.

empty

errorState

Whether the input is in an error state.

focused

Whether the input is focused.

stateChanges

Stream that emits whenever the state of the input changes such that the wrapping MdFormField needs to run change detection.

value

The input element's value.

Methods
