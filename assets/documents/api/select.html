API reference for Angular Material select

import {MatSelectModule} from '@angular/material';

Directives

MdSelectTrigger

Allows the user to customize the trigger that is displayed when the select has a value.

Selector: md-select-trigger

MdSelect

Selector: md-select
Exported as: mdSelect, matSelect
Properties
NameDescription
@Input('aria-label')

ariaLabel

Aria label of the select. If not specified, the placeholder will be used as label.

@Input('aria-labelledby')

ariaLabelledby

Input that can be used to specify the aria-labelledby attribute.

@Input()

compareWith

A function to compare the option values with the selected values. The first argument is a value from an option. The second is a value from the selection. A boolean should be returned.

@Input()

disableRipple

Whether ripples for all options in the select are disabled.

@Input()

floatPlaceholder

Whether to float the placeholder text.

@Input()

multiple

Whether the user should be allowed to select multiple options.

@Input()

panelClass

Classes to be passed to the select panel. Supports the same syntax as ngClass.

@Input()

placeholder

Placeholder to be shown if no value has been selected.

@Input()

required

Whether the component is required.

@Input()

value

Value of the select control.

@Output()

change

Event emitted when the selected value has been changed by the user.

@Output()

onClose

Event emitted when the select has been closed.

@Output()

onOpen

Event emitted when the select has been opened.

customTrigger

User-supplied override of the trigger element.

optionGroups

All of the defined groups of options.

optionSelectionChanges

Combined stream of all of the child options' change events.

options

All of the defined select options.

overlayDir

Overlay pane containing the options.

panelOpen

Whether or not the overlay panel is open.

selected

The currently selected option.

trigger

Trigger that opens the select.

triggerValue

The value displayed in the trigger.

Methods
close

Closes the overlay panel and focuses the host element.

focus

Focuses the select element.

open

Opens the overlay panel.

toggle

Toggles the overlay panel open or closed.
