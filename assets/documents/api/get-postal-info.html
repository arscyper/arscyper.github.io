API reference for Interactive Geocoding - Find postal info

{
  "preferences": {
    "maxReturnedCandidates": 10
  },
  "address": {
    "mainAddressLine": "Albany"
  }
}

Preferences

Parameter Default Description
maxReturnedCandidates 10 Max number of possible candidates in a response.

Address

Parameter type Description
mainAddressLine string A value to use for search criteria. It could possibly be address, place name post code etc.
