API reference for Angular Material list

import {MatListModule} from '@angular/material';

Directives

MdListDivider

Divider between items within a list.

Selector: md-divider

MdList

A Material Design list component.

Selector: md-list md-nav-list

MdListItem

An item within a Material Design list.

Selector: md-list-item a[md-list-item]

MdListOption

Component for list-options of selection-list. Each list-option can automatically generate a checkbox and can put current item into the selectionModel of selection-list if the current item is checked.

Selector: md-list-option
Properties
NameDescription
@Input()

checkboxPosition

Whether the label should appear before or after the checkbox. Defaults to 'after'

@Input()

disabled

Whether the option is disabled.

@Input()

selected

Whether the option is selected.

@Input()

value

Value of the option

@Output()

deselected

Emitted when the option is deselected.

@Output()

destroyed

Emitted when the option is destroyed.

@Output()

selectChange

Emitted when the option is selected.

onFocus

Emitted when the option is focused.

Methods
focus

Allows for programmatic focusing of the option.

toggle

Toggles the selection state of the option.

MdSelectionList

Material Design list component where each item is a selectable option. Behaves as a listbox.

Selector: md-selection-list
Properties
NameDescription

options

The option components contained within this selection-list.

selectedOptions

The currently selected options.

Methods
deselectAll

Deselects all of the options.

focus

Focus the selection-list.

selectAll

Selects all of the options.
