API reference for Interactive Geocoding - Administrative Area

{
  "preferences": {
    "maxReturnedCandidates": 10,
    "clientCoordSysName" : "EPSG:4326",
    "originXY" : [-73.7563842, 42.6560116],
    "distance" : 5.0,
    "distanceUnits" : "MILES",
    "customPreferences" : {
      "COMPRESSED_AREA_RESULT" : "true"
    }
  },
  "address": {
    "mainAddressLine": "Alb",
    "country" : "USA"
  }
}

Preferences

Parameter Default Description
maxReturnedCandidates 10 Max number of possible candidates in a response.
clientCoordSysName epsg:4326 Coordinate system that should be used for points.
originXY null X,Y coordinate that should be used to start the search from
distance 5 distance used to search around the originXY.
distanceUnits MILES Distance unit for handling distance value appropriately.
customPreferences null Additional preferences that can passed in as key: value

customPreferences

{
  "preferences": {
    "customPreferences" : {
      "COMPRESSED_AREA_RESULT" : "true"
    }
  }
}
Parameter Default Description
COMPRESSED_AREA_RESULT false allows search to be compressed to administrative area.

Address

Parameter Type Description
mainAddressLine string A value to use for search criteria. It could possibly be single line address, place name post code etc.
