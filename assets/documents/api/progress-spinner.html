API reference for Angular Material progress-spinner

import {MatProgressSpinnerModule} from '@angular/material';

Directives

MdProgressSpinner

component.

Selector: md-progress-spinner
Properties
NameDescription
@Input()

mode

Mode of the progress circle

Input must be one of the values from ProgressMode, defaults to 'determinate'. mode is bound to the host as the attribute host.

@Input()

strokeWidth

Stroke width of the progress spinner. By default uses 10px as stroke width.

@Input()

value

Value of the progress circle. It is bound to the host as the attribute aria-valuenow.

MdSpinner

component.

This is a component definition to be used as a convenience reference to create an indeterminateinstance.
