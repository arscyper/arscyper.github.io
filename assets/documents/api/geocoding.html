API reference for Geocoding

{
  "type": "ADDRESS",
  "preferences": {
    "returnAllCandidateInfo": true
  },
  "addresses": [
    {
      "mainAddressLine": "1 Global View Troy NY 12180",
      "country": "USA"
    }
  ]
}

Preferences

Parameter Default Description
returnAllCandidateInfo false Return all information about a found candidate.

Address

Parameter type Description
mainAddressLine string Single line address to find location information.
country string country value where to look for a particular address.
