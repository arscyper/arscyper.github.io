API reference for Geocoding - Reverse

{
  "preferences": {
    "returnAllCandidateInfo": true,
    "clientCoordSysName": "epsg:4326"
  },
  "points": [
    {
      "country": "USA",
      "geometry": {
        "type": "point",
        "coordinates": [
          -73.6863824,
          42.700061
        ]
      }
    }
  ]
}

Preferences

Parameter Default Description
returnAllCandidateInfo false Return all information about a found candidate.

Points

Parameter type Description
country string country value where to look for a particular address.
geometry GeoPos Location point with coordinate system information.

GeoPos

Parameter Type Description
type string Type of coordinates. Possible values point
coordinate array any array containing X,Y coordinates.
crs Crs object containing coordinate system

Crs

Parameter Type Description
type string Type of Crs. Possible values name
properties Properties Properties for Crs description.

Properties

Parameter Type Description
name string description of Crs. i.e. epsg:4326
