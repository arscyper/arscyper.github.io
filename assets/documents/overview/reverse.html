API Overview for Geocoding - Reverse

Simple Reverse Geocoding

Using X, Y coordinates to find location information

URL 

http://localhost:8082/Geocode/rest/GeocodeService/reverseGeocode

POST content 

{
  "preferences": {
    "returnAllCandidateInfo": true,
    "clientCoordSysName": "epsg:4326"
  },
  "points": [
    {
      "country": "USA",
      "geometry": {
        "type": "point",
        "coordinates": [
          -73.6863824,
          42.700061
        ]
      }
    }
  ]
}
