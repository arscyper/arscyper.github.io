API Overview for Interactive Geocoding - Find postal info

Find Postcode

Interactive API allow users to find information about post codes using single line address. It can be accomplished using only mainAddressLine.

URL 

http://localhost:8082/Geocode/rest/GeocodeService/interactive

POST request 

{
  "preferences": {
    "maxReturnedCandidates": 10
  },
  "address": {
    "mainAddressLine": "Albany"
  }
}

GET request 

http://localhost:8082/Geocode/rest/GeocodeService/interactive?mainAddress=Albany&maxCands=10
