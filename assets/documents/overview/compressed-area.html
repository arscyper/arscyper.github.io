API Overview - Administrative Area Search

Administrative Area

The api can enable looking up of only administrative areas for example auto completing a field based off City, Post code or State,Province

URL 

http://localhost:8082/Geocode/rest/GeocodeService/interactive

POST request 

{
  "preferences": {
    "maxReturnedCandidates": 10,
    "clientCoordSysName" : "EPSG:4326",
    "originXY" : [-73.7563842, 42.6560116],
    "distance" : 5.0,
    "distanceUnits" : "MILES",
    "customPreferences" : {
      "COMPRESSED_AREA_RESULT" : "true"
    }
  },
  "address": {
    "mainAddressLine": "Alb"
  }
}

GET request 

http://localhost:8082/Geocode/rest/GeocodeService/interactive?
mainAddress=Albany&country=USA
&originXY=-73.7563842,42.6560116
&distance=5&distanceUnits=MILES&maxCands=10

Restrict Area using originXY

Search result can be restricted to a particular area by providing originXY for which the API will only return results around a given originXY and it will not go beyond the provided distance value.
