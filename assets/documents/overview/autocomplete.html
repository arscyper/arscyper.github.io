API Overview for Interactive Geocoding - Autocomplete

Simple Autocomplete

For creating simple input field for autocomplete it does not require big payload for request. It can be accomplished using only mainAddressLine. Also, if you want to control how many candidates are returned then you could also provide maxReturnedCandidates. This would allow any app to accomplish autocomplete for single line address.

URL 

http://localhost:8082/Geocode/rest/GeocodeService/interactive

POST content 

{
  "preferences": {
    "maxReturnedCandidates": 10
  },
  "address": {
    "mainAddressLine": "Albany"
  }
}
