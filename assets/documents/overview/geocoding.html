API Overview for Geocoding

Simple Geocoding

For simple geocoding full address could be provided in single line format.

URL 

http://localhost:8082/Geocode/rest/GeocodeService/geocode

POST content 

{
  "type": "ADDRESS",
  "preferences": {
    "returnAllCandidateInfo": true
  },
  "addresses": [
    {
      "mainAddressLine": "1 Global View Troy NY 12180",
      "country": "USA"
    }
  ]
}

GET request 

http://localhost:8082/Geocode/rest/GeocodeService/geocode?
mainAddress=1+Global+View+Troy+NY+12180&country=USA
