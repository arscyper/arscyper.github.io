API Overview - Restrict Area

Restrict Area

Search area can restricted using either preferences or address values. For example if country field is provided the api will only return results from that particular country. This would allow any app to accomplish autocomplete for single line address.

URL 

http://localhost:8082/Geocode/rest/GeocodeService/interactive

POST request 

{
  "preferences": {
    "maxReturnedCandidates": 10,
    "clientCoordSysName" : "EPSG:4326",
    "originXY" : [-73.7563842, 42.6560116],
    "distance" : 150.0,
    "distanceUnits" : "METERS"
  },
  "address": {
    "mainAddressLine": "Alb",
    "country" : "USA"
  }
}

GET request 

http://localhost:8082/Geocode/rest/GeocodeService/interactive?
mainAddress=Albany&country=USA
&originXY=-73.7563842,42.6560116
&distance=150&distanceUnits=METERS&maxCands=10

Restrict Area using originXY

Search result can be restricted to a particular area by providing originXY for which the API will only return results around a given originXY and it will not go beyond the provided distance value.

Restrict Area using country

Search results can be restricted to a particular country using the country field of address object.
